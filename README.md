## Installation

```bash
# Basic install
pip install rapmat

# MatterSim support
pip install rapmat[mattersim]

# NequIP support
pip install rapmat[nequip]

# uPET support
pip install rapmat[upet]

# All calculator backends at once
pip install rapmat[all-calculators]
```

## Usage

Just run it's TUI:

```bash
rapmat
```
